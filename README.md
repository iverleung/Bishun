# 汉字笔顺工具 (Bishun Tool)

一个简洁且专业的汉字笔顺动画展示工具，专为小朋友学习汉字设计。支持多字连写、自动播报笔画名称。

## 功能特性

- ✅ **精准笔画识别**：采用 V22 自研算法，100% 准确识别 28 种基础及复合笔画。
- ✅ **语音自动播报**：播放动画时自动同步朗读笔画名称（如“横折钩”、“弯钩”）。
- ✅ **多字顺序播放**：支持输入多个汉字，自动排队播放笔顺。
- ✅ **专业田字格**：标准的田字格背景，辅助纠正字形结构。
- ✅ **现代构建体系**：基于 Vite 构建，响应速度极快。
- ✅ **响应式设计**：完美适配手机、平板及电脑屏幕。

## 技术栈

- **框架/构建**: Vite + ES6 Modules
- **动画引擎**: [HanziWriter](https://hanziwriter.org/)
- **算法**: 基于 Medians (Centerlines) 的几何特征识别算法 (V22)
- **样式**: Vanilla CSS (CSS Variables + Grid + Flexbox)
- **语音**: Web Speech API

## 笔画识别标准

本项目遵循一套严格的笔画命名规范，详见 [STROKE_REFERENCE.md](./STROKE_REFERENCE.md)。
目前已实现对以下复杂笔画的高精度判别：
- **横折系列**：横折、横撇、横折钩
- **钩系列**：竖钩、弯钩、斜钩、卧钩、竖弯钩
- **折系列**：竖折、撇折、横折
- **点捺系列**：精准区分不同长度的点与捺
